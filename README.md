# ✝️ Ambassador of Christ — Website

A modern, mobile-responsive website for the **Ambassador of Christ** online Christian community. Built to share the Word of God, connect believers globally, and expand into merchandise and outreach.

---

## 📌 Project Overview

| Field | Details |
|---|---|
| **Project Name** | Ambassador of Christ Website |
| **Purpose** | Online Christian community platform |
| **Stack** | HTML, CSS, Vanilla JavaScript (single file) |
| **Status** | Phase 1 Complete — Ready for deployment |

---

## 📄 Pages & Sections

- **Home** — Hero banner with scripture verse and call-to-action
- **About** — Ministry vision, values, and mission
- **Blog** — Weekly devotionals and articles
- **Videos** — Embedded video message gallery
- **Community** — Member testimonies
- **Events** — Upcoming gatherings and announcements
- **Store** — Faith merchandise (coming soon)
- **Contact** — Message form and social links

---

## ✨ Features

- ✅ Fully mobile responsive
- ✅ Smooth scroll animations
- ✅ Sticky navigation bar
- ✅ Newsletter signup
- ✅ Contact form
- ✅ Social media links (YouTube, TikTok, Instagram, Twitter/X, Facebook)

## 🚀 Getting Started

1. Download `ambassador-of-christ.html`
2. Open in any browser — no build tools or dependencies needed
3. To deploy, upload the file to any web host (Netlify, Vercel, GitHub Pages, etc.)
